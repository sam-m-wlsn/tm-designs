
# Static site compiler for the first iteration of Tyler's portfolio site


## Paths
file type | source | destination
--- | --- | ---
Styles | `./src/scss/*` | `./dist/css/*`
Javascript | `./src/js/*` | `./dist/js/*`
HTML | `./src/html/*` | `./dist/*`

## Commands
Task | Command 
--- | ---
Install | `npm i`
Build Distributables |  `npm build`
